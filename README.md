## 📌 About the Project

This is a modern, responsive **developer portfolio website** built to highlight:

* 👨‍💻 My technical skills
* 💼 Work experience & internships
* 🚀 Projects with real-world impact
* 📬 Easy ways to connect with me

The design focuses on **clean UI, smooth animations, and strong user experience**.

---

## ✨ Features

* ⚡ Smooth scrolling navigation
* 🎨 Modern UI with gradient styling
* 📱 Fully responsive (mobile + desktop)
* 🧠 Animated sections using Intersection Observer
* 🖼️ Interactive project cards with image preview slider
* 📥 Resume download option
* 📬 Contact section with social links
